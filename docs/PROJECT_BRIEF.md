# MAISON OTTO - Proje Özeti

## Proje Amacı

MAISON OTTO markası için premium, modern ve minimal bir erkek giyim e-ticaret web sitesi geliştirilecektir.

Proje şu an yalnızca frontend olarak geliştirilecektir.

Ürünler mock data üzerinden çalışacaktır.

Ancak mimari ileride:

* Backend entegrasyonu
* Ödeme sistemleri
* Stok yönetimi
* Sipariş yönetimi
* Admin paneli

eklenebilecek şekilde hazırlanmalıdır.
